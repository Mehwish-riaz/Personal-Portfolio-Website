# 🌐 Personal Portfolio Website

A modern **personal portfolio website** built with **HTML5, CSS3, Bootstrap 5**, showcasing my journey as a **Software Engineering student** and a **Frontend Web Developer in progress**.  
This project includes responsive design, skill animations, project showcases, education & certificates section, contact form, and a modern UI inspired by professional portfolios.

---

# 🚀 Live Demo
https://personal-portfolio-website-peach-mu.vercel.app/

---

# 📌 Features

## 👩‍💻 About Me
- Display of personal details, professional title, and a short bio.
- Modern dark-blue theme with **white & blue highlights**.
- Sticky navbar with transparent overlay effect.

## 🖥️ Skills Section
- Display of technical and software skills.
- Animated **progress bars** indicating proficiency.
- Left-right split layout for better visual balance.

## 📂 Projects Section
- Showcase of personal projects with **card design**.
- Interactive hover effects with shining navy-blue borders.
- Each project includes title, description, thumbnail, and GitHub link.

## 🎓 Education & Certificates
- Education timeline showing institutes and achievements.
- List of certificates with clickable links.
- Color-coded sections for clarity (white for education, blue for certificates).

## 📫 Contact Section
- Contact form with fields styled in navy-blue glowing borders.
- Social media icons: **LinkedIn, GitHub, Email**.
- Professional footer with copyright.
